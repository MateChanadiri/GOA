the five programming languages

1)python

python is the easiest language in coding also it is a high level general purpose programming language its design philosophy emphasizes code readibility with the use of significant indentation.

2)Javascript

JavaScript often known as JS is a programming language and core technology of the Web, alongside HTML and CSS. its not a hard language but its not an easy one!

3)java

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

4)c#

C# is a general-purpose high-level programming language supporting multiple paradigms. C# encompasses static typing, strong typing, lexically scoped, imperative, declarative, functional, generic, object-oriented, and component-oriented programming disciplines.

5) ruby

Ruby is an interpreted, high-level, general-purpose programming language. It was designed with an emphasis on programming productivity and simplicity
